# ProCayThue Roblox - Landing Page

Trang web giới thiệu dịch vụ cày thuê Roblox với đầy đủ:

- Hero section + form tư vấn nhanh.
- Các gói dịch vụ, bảng giá và công cụ ước tính chi phí.
- Quy trình làm việc, đánh giá khách hàng, FAQ.
- Form liên hệ và bố cục responsive cho mobile.

## Cách chạy local

```bash
python3 -m http.server 8000
```

Sau đó mở `http://localhost:8000`.
